# О проекте
Это простое Windows Forms-приложение на C#, которое шифрует и дешифрует текст с помощью алгоритма TripleDES. Написал в рамках самостоятельной работы по криптографии.
Что умеет программа
Ввести любой текст в поле textBox1
Зашифровать его кнопкой «Зашифровать» — результат появится в textBox2
Автоматически дешифровать обратно — результат в textBox3
Использует симметричное шифрование: TripleDES, режим ECB, паддинг PKCS7
# Технологии
Язык: C# (.NET Framework)
Платформа: Windows Forms
Криптография: System.Security.Cryptography
Алгоритм: TripleDES (ECB, PKCS7)
# Как запустить
Откройте решение в Visual Studio 2019 или 2022
Убедитесь, что целевая платформа — .NET Framework 4.7.2 или новее
Нажмите F5 или кнопку «Запуск»
Введите текст, нажмите кнопку шифрования, смотрите результат
# Важно: ключ шифрования генерируется заново при каждом запуске и хранится только в памяти. После перезапуска программы расшифровать старые данные не получится!
Известные ограничения (и что я понял в процессе)
Не используйте ECB в реальных проектах — преподаватель сказал, что это небезопасно, но для учебной работы сойдёт.
Encoding.Default может некорректно работать с кириллицей на разных системах. Лучше бы UTF8, но я пока не стал усложнять.
При дешифровании используется str.Length для размера буфера — это не совсем правильно, но в рамках теста работает.
Нет обработки исключений — если ввести очень много текста или случится ошибка криптографии, приложение просто упадёт.
Это учебный проект. Не используйте этот код для защиты реальных данных!
